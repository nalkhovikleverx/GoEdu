# Solution Space

An ad-hoc Event-Storming located [here](https://miro.com/app/board/uXjVK2jettQ=/).

Here, we identify several bounded contexts designed to address the domain problem space:

- **User Management and Access**:
  A generic subdomain focused on registrations and access control.

- **Profile Management**:
  A core subdomain responsible for managing operations related to Consultants and Clients.

- **Administration**:
  A supportive subdomain handling consultant verification processes.

- **Consultation Management**:
  A core subdomain dedicated to managing consultations.

- **Online Meeting Management**:
  A core subdomain overseeing the online consultation process.
